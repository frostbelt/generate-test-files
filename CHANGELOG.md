# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2024-01-04

### Added
- Initial release of generate-test-files CLI tool
- Support for text, binary, JSON, HTML, CSV, XML, and zero-filled files
- Support for image files: JPEG, PNG, GIF, WebP, BMP
- Support for video files: MP4, AVI, MOV, MKV, WebM
- Flexible file size specification (B, KB, MB, GB, TB)
- Batch file generation with automatic numbering
- Custom output directory support
- Programmatic API for Node.js integration
- Smart file naming with timestamps and sequence numbers
- Comprehensive documentation and examples

### Features
- Fast file generation using Node.js streams
- China timezone support (UTC+8)
- Automatic file extension detection
- Error handling and validation
- CLI help and usage information
